<Main Success Scenario>
1. 選完照片，讓USER選擇 "自動排版" 或 "手動排版"。 (自動排版就是原來的加入編輯區，手動排版就是只加入暫存區)
2. 選擇 "自動排版" 或 "手動排版"後，出現一個文字框讓USER填入標題，並讓USER選擇要套用的主題(Theme)。
(標題旁邊有一個check box “稍後再填”，讓USER可以現在填入標題，同時也提供彈性讓USER選擇稍後再填，而不會在填標題這一步卡住USER。
主題(Theme)這個字，USER比較容易理解，也可與類別區隔。
主題目前包括: 旅遊、Baby、喜宴、生日、美食、聚會、競賽、表演、繪本、食譜、其他)
3. 回到編輯畫面。如果USER是選擇 "自動排版"，請提示USER是否要進行 “預覽播放”。
(並提示USER預覽播放的過程中也可以隨時進行修改。因為會選擇 “自動排版” 的USER，很有可能是希望可以在短時間內就完成編輯的USER)

切換三種顯示及編輯模式: 預覽編輯模式、快速瀏覽模式、全銀幕瀏覽模式
上一頁/下一頁
刪除此頁
BACK/NEXT
編輯程序顯示及切換: "新增圖片", "編輯", "設定", "發佈"
打開編輯功能選單 [002_EditMenu.htm]
復原編輯
Support Resize, and Move(可以room-in, 然後選擇顯示範圍)
貼相片標籤)

(快速瀏覽模式讓USER在畫面上可以同時看到多個頁面，讓USER可以進行排序、選擇縮圖、或是快速刪除、快速貼相片標籤。
快速瀏覽模式之下有以下的功能控制選項:

<Edit(頁面留白時底色的選擇)>
當頁面留白時，(當照片沒有填滿所選擇版面時) 留白版面圖案及顏色=>跟著Theme被自動決定

<Edit (版面比例、Picture Size、Position)>
1. 支援固定幾種的水平:垂直比例 (eg. 16:9, 3:2, 4:3, 1:1, 3:4, 2:3), 這個版面比例整本書是一致的。
(不支援自訂水平:垂直比例)
2. 根據創作者第一次上傳的照片中，那一種水平:垂直比例的照片數量最多(或最接近)，default選擇為本書的水平:垂直比例。但創作者可以重新挑選其他系統支援的水平:垂直比例。
3. Default的大小及位置為: 選擇fit width或fit height其中一種，將整張圖片完整置入，不裁切。如果有空白就水平或垂直置中。
4. 如果創作者對於Default的大小不滿意，可以重新選擇fit width或fit height。(只有兩種選擇)
5. 如果創作者對於Default的位置不滿意，fit width的情況下可以垂直調整，fit height的情況下可以水平調整。如果圖片已經同時fit width及fit height，意即這張圖片與本書的版面完全一致，則位置無法調整。
(不支援透過位置調整，以達到裁切的目的==>
a. 如果原本的大小選擇為可以顯示完整照片，位置的調整，照片邊緣觸碰到畫板邊緣就停止，不裁切照片。
b. 如果原本的大小選擇為照片會填滿畫面，位置的調整，照片邊緣觸碰到畫板邊緣就停止，不允許因為位置移動過多而出現邊緣的留白)
6. 位置調整也可以讓USER直接輸入位移的pixel。(如果創作者需要前後張對齊，就可以直接輸入數字)
7. USER picture resolution不足時，強迫放大。(不管實際pixel,只管長寬比例)

<Edit (照片上傳數量限制相關問題)>
1. 暫存區是跨書本共用。
2. 暫存區照片數量 => 小於等於400張
3. 編輯區頁數限制 (或是發佈頁數限制) => 小於等於50張
4. 未發佈之書籍何時被刪除 => 連續兩個月未編輯此書
(如果user在兩個月到期前，又編輯了此書，則又可延長兩個月)
5. 暫存區之照片何時被刪除 => 連續三個月未使用編輯的功能
6. 未發佈+已發佈照片書數目限制 => 十本 (得付費以增加此數目)

<Notes>
支援10次復原 (可以往前復原10步)

Tag相關的行為及隱私權的問題, 個人想法如下:
=> a. Tag friends(標註朋友)的行為多發生在照片書(繪本較少)，目的是增加社交的互動。
=> b. Tag friends屬於編輯的行為之一，此外tag friends也會影響到書本分享的範圍，所以只有創作者才能對作品Tag friends.
=> c. 創作者Tag friends之後，被Tag的人會收到通知。
=> d. 被Tag的人，可以審核被Tag這件事，是否被發佈在自己的塗鴉牆上。(如同在FB上被朋友標註在一張照片上) 也可以不進行審核預設同意。
=> e. 被Tag的人，可以remove Tag。
=> f. 被Tag的人，對於整本書都有閱讀的權利。(remove Tag這項權利就消失)
=> g. Default設定被Tag的人的朋友也有同樣閱讀整本書的權利 (創作者可以disable this item)

關於類別與主題(Theme), 我們之前一致的認知應該是如下:
1. 類別代表的是這本書屬性上的分類。例如這本書屬於繪本。
2. 主題(Theme)是這本書想要套用的播放方式，背景音樂，第一頁及最後一頁的排版方式等等與內容編輯及播放相關的設定。
3. 類別與主題可能是independent。例如內容是美食，但創作者可以套用旅遊的Theme。
